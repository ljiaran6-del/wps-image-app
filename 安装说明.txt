WPS图片复刻版 安装说明
======================

这份说明是给不太懂技术的朋友准备的。
只要按照下面的步骤一步步来，应该就能在另一台 Windows 电脑上跑起来。


一、你需要先准备什么
--------------------

1. 一台 Windows 10 或 Windows 11 的电脑
2. 能正常上网（因为AI功能需要联网调用接口）
3. 一个 OpenAI 的 API Key（或者第三方代理的 API Key）

如果你不知道什么是 API Key，可以先看下面的"常见问题"。


二、安装 Python
---------------

这个工具是用 Python 写的，所以电脑上要先安装 Python。

步骤：

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击页面上最大的黄色按钮，下载最新版的 Python
3. 双击下载好的安装文件
4. 非常重要：在安装界面下方，勾选 "Add Python to PATH"（把Python加到环境变量）
5. 点击 "Install Now" 安装
6. 等待安装完成，点击 Close

怎么确认安装好了？
- 按键盘上的 Win + R
- 输入 cmd，按回车
- 在黑色窗口里输入 python --version
- 如果看到类似 Python 3.11.4 的字样，说明安装成功


三、把项目放到电脑上
--------------------

把整个 wps-image-app 文件夹复制到目标电脑上。

可以这样做：
- 用 U 盘拷贝
- 用微信/QQ/邮件发送压缩包，然后解压
- 用百度网盘、阿里云盘等工具同步

复制完成后，记住这个文件夹的位置，比如：
C:\Users\你的名字\Desktop\wps-image-app


四、安装依赖
------------

依赖就是这个工具需要用到的其他程序包。我们已经写好了清单，让电脑自动安装就行。

步骤：

1. 按 Win + R，输入 cmd，按回车，打开黑色命令窗口
2. 输入下面的命令，进入项目文件夹（注意替换成你自己的路径）：

   cd /d C:\Users\你的名字\Desktop\wps-image-app

3. 输入下面的命令，创建虚拟环境（这一步只需要做一次）：

   python -m venv venv

   等几秒钟，你会看到项目文件夹里多出来一个叫 venv 的文件夹。

4. 输入下面的命令，安装依赖：

   .\venv\Scripts\pip install -r requirements.txt

   这一步会下载一些东西，根据网速可能需要 1-5 分钟。看到最后一行是 "Successfully installed" 就是成功了。


五、配置 API Key
----------------

AI 功能需要调用 OpenAI 的接口，所以需要配置 API Key。

步骤：

1. 打开项目文件夹里的 config.json 文件
   （可以用记事本打开）

2. 你会看到类似这样的内容：

   {
       "BASE_URL": "",
       "API_KEY": "",
       "MODEL": "gpt-image-2"
   }

3. 把 BASE_URL 改成你的代理地址或 OpenAI 官方地址
   把 API_KEY 改成你的 Key

   例如：

   {
       "BASE_URL": "https://api.openai.com/v1",
       "API_KEY": "sk-xxxxxxxxxxxxxxxxxxxxxxxx",
       "MODEL": "gpt-image-2"
   }

4. 保存并关闭文件

如果你用的是第三方代理，BASE_URL 和 API_KEY 要按代理商给你的来填。


六、启动工具
------------

有两种方式可以启动：

方式一：双击启动
直接双击项目文件夹里的 start.bat 文件。

方式二：命令行启动
1. 按 Win + R，输入 cmd，按回车
2. 输入：

   cd /d C:\Users\你的名字\Desktop\wps-image-app\src
   ..\venv\Scripts\python.exe main.py


七、第一次使用
--------------

启动后，你会看到一个深色窗口。

1. 点击左上角 "文件" -> "打开"，选择一张图片
2. 点击左侧工具栏的按钮切换功能
3. AI 功能需要先框选或涂抹区域，再点右侧的"开始"按钮
4. 标注功能在右侧可以调颜色、粗细、字号


八、常见问题
------------

Q1：双击 start.bat 后闪了一下就关了，怎么办？
A：可能是依赖没装好，或者 config.json 配置不对。建议用"方式二"命令行启动，这样能看到具体报错信息。

Q2：提示 "未设置 OpenAI API Key" 怎么办？
A：检查 config.json 里的 API_KEY 是否填对了，保存后重新启动。

Q3：AI 功能一直转圈，没有结果？
A：AI 调用需要联网，而且比较慢，通常要等 5-15 秒。如果超过 30 秒还没结果，可能是网络问题或 API Key 余额不足。

Q4：什么是 API Key？从哪里获取？
A：API Key 就像是调用 AI 服务的密码。你可以：
   - 去 OpenAI 官网申请（需要国外信用卡）
   - 在国内找一些 OpenAI 代理服务（搜索 "OpenAI API 代理"）
   - 让会技术的朋友帮你买一个

Q5：安装依赖时提示 pip 不是内部命令？
A：说明你安装 Python 时没有勾选 "Add Python to PATH"。卸载 Python 重新安装，记得勾选那个选项。

Q6：这个工具收费吗？
A：工具本身不收费，但 AI 功能会消耗你的 API Key 余额。每次调用大概几分钱到几毛钱不等。


九、如果还是搞不定
------------------

请把下面这些信息发给会技术的朋友：

1. 你的 Windows 版本（如 Windows 11）
2. Python 版本号（cmd 里输入 python --version 得到的结果）
3. 启动时黑色窗口里的报错文字（如果有的话）
4. config.json 里的 BASE_URL 和 MODEL 填了什么（不要把 API_KEY 发给别人）


祝你使用愉快！
