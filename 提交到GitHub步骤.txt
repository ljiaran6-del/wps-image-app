提交到 GitHub 的步骤
====================

这个文件夹里的内容已经整理好了，可以直接提交到 GitHub。


第一步：注册/登录 GitHub
--------------------------

1. 打开 https://github.com
2. 注册一个账号（如果还没有的话）
3. 登录


第二步：创建新仓库
------------------

1. 点击右上角 + 号，选择 "New repository"
2. Repository name 填：wps-image-app
3. Description 可以填：轻量级 Windows 桌面图片处理工具，复刻 WPS图片 的 AI 编辑和标注功能
4. 选择 Public（公开）或 Private（私有）
5. 不要勾选 "Initialize this repository with a README"（因为我们已经准备好了 README.md）
6. 点击 "Create repository"


第三步：把本地代码推送到 GitHub
--------------------------------

创建完仓库后，GitHub 会显示类似下面的命令。在你的电脑上操作：

1. 按 Win + R，输入 cmd，按回车
2. 进入 .github-prep 文件夹：

   cd /d C:\Users\WD\wps-image-app\.github-prep

3. 依次输入下面的命令：

   git init
   git add .
   git commit -m "初始提交：WPS图片复刻版 v1.0"
   git branch -M main
   git remote add origin https://github.com/你的用户名/wps-image-app.git
   git push -u origin main

4. 输入你的 GitHub 用户名和密码/Token

如果命令都成功执行，刷新 GitHub 页面，就能看到代码了。


第四步：完善仓库信息（可选但建议做）
------------------------------------

1. 在 GitHub 仓库页面，点击 "Add topics"
   可以添加标签，比如：python, pyside6, image-editor, ai, openai, inpainting

2. 点击 "Releases" -> "Create a new release"
   - Tag version: v1.0.0
   - Release title: WPS图片复刻版 v1.0.0
   - 描述里写主要功能和修复

3. 上传截图：
   - 把应用截图拖到 Issue 或 README 编辑框里
   - GitHub 会自动生成图片链接
   - 把链接填到 README.md 的截图位置


第五步：告诉别人你的项目
------------------------

1. 复制仓库地址：https://github.com/你的用户名/wps-image-app
2. 分享到朋友圈、技术群、论坛等
3. 如果有人提 issue 或 PR，及时回复


注意事项
--------

- 这个文件夹里的 config.json 是空的模板，没有 API Key，可以放心提交
- .gitignore 已经配置好，venv、__pycache__、config.json 都不会被提交
- 如果你之后修改了代码，可以重复 "git add ."、"git commit"、"git push" 来更新
