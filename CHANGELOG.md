# 更新日志

## [1.0.0] - 2026-07-12

### 新增

- AI消除：支持矩形选区和涂鸦蒙版
- AI去水印：针对水印场景优化的消除
- AI局部改图：框选区域 + prompt 修改
- AI扩图：支持预设比例和自定义目标尺寸
- 标注工具：画笔、箭头、矩形、圆形、文字、马赛克、高斯模糊
- 基础看图器：打开、缩放、拖动、适应窗口、实际大小
- 文件夹浏览：上一张、下一张
- 保存/另存为：支持 PNG/JPG/WEBP/BMP
- 撤销/重做：默认 50 步历史
- API Key 设置对话框
- 深色主题 UI

### 修复

- 修复 AI消除 调用 API 时 mask 格式错误
- 修复涂鸦蒙版无法显示的问题
- 修复标注画笔绘制后位置偏移
- 修复 AI消除 结果扩散到 mask 外的问题（增加后处理融合）
- 修复代理返回 b64_json 时的兼容性问题
- 修复深色主题下部分控件文字不可见
- 修复左侧工具栏按钮文字被截断

### 优化

- mask 缩放改用 NEAREST，保持硬边缘
- AI 服务同时兼容 url 和 b64_json 返回
- 自动适配 OpenAI image edit API 支持的输出尺寸
